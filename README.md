Introduction

[PYICUB-LAB](https://github.com/s4hri/pyicub-lab) is a development platform
for coding iCub applications in minutes!

```
git clone https://github.com/s4hri/pyicub-lab
cd pyicub-lab
bash go
```

Using REST API
-------------

```
git clone https://github.com/s4hri/pyicub-lab
cd pyicub-lab
bash go --profile restapi
```
